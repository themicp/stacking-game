## Arduino Setup
* Download LedControl library https://github.com/wayoda/LedControl/releases
* Unzip the file
* Rename ``LedControl-<version>`` folder to ``LedControl``
* Move the ``LedControl`` folder to your Arduino IDE installation libraries folder
* Restart Arduino IDE
* Import ``game.ino``
